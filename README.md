# My Progress So Far:

## Pong (Completed) ✅

- A simple arcade game, you have a ping pong ball and you play against the Enemy AI. If the ping pong ball touches you or the enemy, it adds up to their score. If it touches either wall the game ends, the score resets, and you have the ability to play again.
- Additionally, you can view a video of this game in the "1st Game Pong" folder. under the name "0714.mp4". Do be aware that you need to download the file raw to view it.
